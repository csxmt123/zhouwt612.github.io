## Preface

Today, I am going to post a tool (Flameshot) for capture the screen in Ubuntu 18.04.

## Installation
### Stpes 1

```
sudo apt-get install flameshot
```

### Stpes 2

Turn to "Setting -> Devices -> keyboard".
Find the "+".
Input:

Name: Flameshot
Command: flameshot gui
Shortcut: Alt + A
